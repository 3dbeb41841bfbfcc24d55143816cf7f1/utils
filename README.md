# utils

A collection of utils for automating the management of these repos.

## clone-all-repos.sh

Usage:

From this directory, run the following in a Bash terminal session:

```bash
cd ..
./utils/clone-all-repos.sh
```

## pull-all-repos.sh

Usage:

From this directory, run the following in a Bash terminal session:

```bash
cd ..
./utils/pull-all-repos.sh
```
